虚拟线程Virtual threads

虚拟线程是一种比传统 Java 线程（OS线程）更轻量的线程实现。与传统线程由操作系统管理不同，虚拟线程由 JVM 管理，多个虚拟线程可以在少量的操作系统线程（平台线程）上运行。虚拟线程的创建、切换和销毁的开销远小于传统线程，同时虚拟线程可以和普通线程一样的使用。
